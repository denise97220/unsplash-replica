# Unsplash Replica

復刻相片共享網站 Unsplash，並還原 Unsplash 的核心功能，包括顯示相關圖片及作者資訊、依關鍵字搜尋圖片、登入登出、收藏圖片等。

## 產品功能

- 使用者可以註冊個人帳號，並使用個人帳號登入，編輯個人資料，包括名稱、帳號、密碼、email、介紹、個人顯示圖片及背景圖片。
- 使用者可以發布自己的文字推文訊息。
- 使用者可以瀏覽其他使用者的推文訊息，對推文新增回覆、按讚。
- 使用者可以追蹤其他使用者或取消追蹤。
- 使用者可以瀏覽其他使用者個人介紹頁面，瀏覽對方的歷史推文、留言、按讚的內容、追蹤中的使用者及追蹤該名使用者的清單。
- 後台管理者可以瀏覽所有使用者以及所有發布的推文，並且管理者能夠刪除推文。
